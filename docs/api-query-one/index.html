    主动-查询-加签名

    请求方式#

    POST请求

    请求URL#

    /payway/order/find

    请求参数#

    字段类型示例说明是否签名
    apiKey字符串"tc"商户请求标识是
    thirdNo字符串"cs12345789"商户订单号(tradeNo和orderNo三选一)传则需要签名
    orderNo字符串"tc12345789"平台订单号(tradeNo和orderNo三选一)传则需要签名
    sign字符串"qwg35y35yu3h35gwasd"签名否

    返回内容#

    字段类型示例说明
    code整型200请求后台返回操作状态，200为成功操作
    msg字符串""对code进行说明
    data----请参考返回事例
    sign字符串"32dew43rttg5gtbgthy6h6h4t6"签名

    date返回内容#

    字段类型示例说明
    thirdNo字符串"1245648978477"商户单号
    orderNo字符串"998154152121156156"otc订单号
    createTime字符串"1678734859487"订单创建时间
    finishTime字符串"1678734859487"订单完结时间
    timestamp字符串"1678734859487"时间戳
    amount字符串"100"下单金额
    quantity字符串"14.6842878"资产数量
    customerPayType字符串"ZFB"支付宝(ZFB)
    customerAccountNo字符串"6228480234564561654"用户银行卡/支付宝/微信账号
    customerAccountName字符串"张三"用户银行卡账户姓名
    status字符串"成功"成功,失败,待确认,初始状态,下单中
    dealerPayType字符串---币商收/付款方式
    dealerAccountNo字符串---币商收付款账户
    dealerAccountName字符串---｜币商收付款账户昵称
    dealerBankName字符串---币商银行卡名称
    dealerNickname字符串---币商挂单昵称
    type字符串“buy”订单的买卖方向buy/sell
    mgs字符串“”订单状态说明
    remark字符串“”bei备注

    该接口查询频率：1分钟/10次
